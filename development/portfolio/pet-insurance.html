Pet-insurance.co.uk is a trading name of E&L Insurance. When joining E&L Insurance the website had just been launched. This meant the designs were already in place. The front-end code however needed some work as the team did not have a front-end developer to assist with this work prior to launch.

My tasks on this project included re-structuring the front-end architecture, making the site responsive to work across multiple platforms and creating new page designs. The site also needed to be setup for white-labelling purposes, adding a new dimension to the architecture of this particular project.
